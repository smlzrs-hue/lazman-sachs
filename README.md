# Lazman Sachs — Global Intelligence Terminal

A very serious, entirely fabricated enterprise business-analytics dashboard for
**Lazman Sachs** (a Sam Lazarus parody of a certain investment bank).

KPI cards, a share-price line chart, a revenue pie chart, a strategic-initiatives
table. Looks legit. Means nothing.

## The secret

Hold **A + S + D + W at the same time** while looking at the dashboard. The revenue
pie chart declassifies itself into **Laz Runner** — a panicked suited banker
sprinting away from the regulators.

| Key | Action |
|-----|--------|
| `W` | Jump (over bears, briefcases, red candles) |
| `S` | Duck (under audits, subpoenas, paper airplanes) |
| `D` | Accelerate (more speed = more profit, more danger) |
| `A` | Decelerate |
| `Esc` | Back to the spreadsheet |
| `W` (when dead) | Try again |

Score is "profit." It scales with how recklessly fast you choose to run.

## Run locally

It's a static site — no build step.

```bash
python3 -m http.server 8000
# open http://localhost:8000
```

All figures are vibes. Not financial advice.
